Mare-NaluX: COMERI's GPU-based variant of Mare-Nalu: Building on the performance portability layer of Kokkos via parallel_for and NGP-based STK field access (host/device).

### Referencing Mare-Nalu(X)
When disseminating technical work that includes Mare-Nalu simulations, please reference the following original Nalu citation:

	Domino, S. "Sierra Low Mach Module: Nalu Theory Manual 1.0", SAND2015-3107W, Sandia National Laboratories Unclassified Unlimited Release (UUR), 2015. https://github.com/NaluCFD/NaluDoc
	
This document can be found under the [Mare-Nalu Documentation](http://nalu.readthedocs.io/en/latest/source/theory/index.html).

### Mare-Nalu Documentation

The Mare-Nalu documentation [WIP] website is located [under Nalu](http://nalu.readthedocs.io/en/latest/).
